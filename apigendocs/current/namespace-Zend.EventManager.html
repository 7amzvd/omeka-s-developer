EventsCapableInterface Interface indicating that an object composes an EventManagerInterface instance.
ListenerAggregateInterface Interface for self-registering event listeners. Classes implementing this interface may be registered by name or instance with an EventManager, without an event name. The {@link attach()} method will then be called with the current EventManager instance, allowing the class to wire up one or more listeners.
Traits Summary
EventManagerAwareTrait A trait for objects that provide events. If you use this trait in an object, you will probably want to also implement EventManagerAwareInterface, which will make it so the default initializer in a ZF2 MVC application will automatically inject an instance of the EventManager into your object when it is pulled from the ServiceManager.
